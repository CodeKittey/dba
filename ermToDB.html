Festlegung der Entitäten

Festlegen der Feldeigenschaften

  • Datentypen: (Zahl, kurzer Text, langer Text, Währung, Datum)
  • Standardwert: Vorbelegte Werte für da Feld
  • Feldgrösse: Anzahl Zeichen max und min für das Feld
  • Domäne: Wertebereich eines Feldes.

Datentypen

Hängen stark von der eingesetzten Datenbank ab. (Hier MySQL)
  • TINYINT (1 Byte), 0 bis 255 oder von -128 bis 127
  • INT (4 Byte), 0 bis 16.777.215 oder von -8.388.608 bis 8.388.607.
  • FLOAT/DOUBLE (4/ 8 Byte).
  • DATE (3 Byte). Datum im Format 'YYYY-MM-DD'. Wertebereich von 01.01.1000 bis 31.12.9999.
  • TEXT (L+2 Bytes).
  • VARCHAR (M Byte). Zeichenkette variabler Länge, Maximum ist M. Wertebereich für M: 0 bis 255.

Domäne

  • Wertebereiche von Attributen
  • Schulnote hat die Domäne 1.0 bis 6.0
  • Besteht aus einem elementaren Datentypen (Integer, Float, String)
  • Domäne: Wertebereich eines Feldes.
